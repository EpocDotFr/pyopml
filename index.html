PyOPML documentation¶

Welcome! This documentation is about PyOPML, a Python package meant to read, manipulate and write OPML 2.0 files.

Prerequisites¶

  • Python 3.6+

Installation¶

From PyPI:

$ pip install pyopml

Locally, after cloning/downloading the repo:

$ pip install .
